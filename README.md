# Vite, Vitest, P5, TypeScript and ESLint Starter

### First thing first
```
npm install
```

### For husky activation
```
npm run prepare
```

### Init the Vite server
```
npm run dev
or
npx vite
```

### Check the test stuff
#### In order to test, open another terminal window and then use the following
```
npm run test
```
